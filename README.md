# IoT Weather Monitoring System

## Overview

This project is an IoT-based Weather Monitoring System developed using ESP8266, DHT11, and a rain sensor. The system collects real-time environmental data such as temperature, humidity, and rainfall levels, and transmits it over Wi-Fi to a cloud dashboard for remote monitoring.

## Features

- Real-time temperature monitoring
- Real-time humidity monitoring
- Rainfall detection and monitoring
- Wireless data transmission using ESP8266
- Cloud-based visualization dashboard
- Remote access to sensor readings

## Hardware Components

- ESP8266 NodeMCU
- DHT11 Temperature and Humidity Sensor
- Rain Sensor Module
- Breadboard
- Jumper Wires
- USB Power Supply

## Software Used

- Arduino IDE
- Blynk IoT Platform
- C++

## Working

The ESP8266 reads temperature and humidity data from the DHT11 sensor and rainfall data from the rain sensor. The collected sensor values are processed and transmitted through Wi-Fi to the Blynk cloud platform. Users can monitor the environmental conditions remotely through a web or mobile dashboard.

## Applications

- Weather Monitoring
- Smart Agriculture
- Environmental Monitoring
- Remote Sensing Systems
- IoT-based Smart Systems
